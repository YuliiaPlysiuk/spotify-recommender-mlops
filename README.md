# Spotify Recommendation System (MLOps)

This project builds a personalized music recommendation system using Spotify API.

## Features
- Fetches real Spotify listening history
- Extracts audio features
- Builds user taste profile
- Recommends unseen songs

## Tech Stack
Python, Spotify API, scikit-learn, FastAPI (later)

## Goal
End-to-end MLOps system with continuous learning recommendations
